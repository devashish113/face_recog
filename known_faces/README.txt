Place photos of authorized/known people in this folder.

NAMING FORMAT:
  person_name.jpg    →  labeled as "Person Name"
  john_doe.jpg       →  labeled as "John Doe"
  devashish.png      →  labeled as "Devashish"

RULES:
  - One clear face per photo
  - Supported formats: .jpg, .jpeg, .png, .bmp
  - Filename (without extension) becomes the person's display name
  - Underscores (_) are replaced with spaces automatically
  - Names are title-cased automatically

The server loads all faces at startup. 
Use POST /reload to reload without restarting.
